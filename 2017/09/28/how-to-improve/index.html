2017年 Q4季度学习计划

引言

早在我大一的时候就看到过这样的一篇文章，如何在一年之内提高自己，当时给我的震撼是无以伦比的，尽管在我大一到大三的这段时间里，见识了许许多多优秀的人给我带来各式各样的体验，但这毕竟是第一次接触到这样“目标-计划-执行-反馈”的模式。

当时的感受真的是，犹如普罗米修斯给人间带来初火般，那时刚刚高考结束，脑袋里一直有些懵懂未开的想法。但初入大学，时间多得犹如回到了童年，这是踏入校园之后从未体验过的，能玩游戏玩到无聊看着太阳发呆。心中积郁很多无名的东西，好像不该这么放纵，因为我已经到了某个年纪了。社会给了大学生不用承担社会责任的四年，究竟是为了什么呢？

思索很多，这也是自高考结束后第一次执笔，真的是把话说出来是一套，把话写出来又是一套，有些话如鲠在喉，还是以后慢慢梳理写出来吧。

清晰的目标

目标

  1. 围绕着b站的机器学习岗位建立知识体系
  2. 围绕着python书建立基本coding experience
  3. 围绕托福考试建立英语体系的建立
  4. 围绕实习导向建立一些择业的了解

时间安排

  • 国庆长假：一个好的开始很重要，前三个目标最好能在长假的时候打下基础，万事开头难，国庆长假需要啃一些东西，适应一些东西。
  • 正常读书时期，每周1-3上课时间，4-7学习时间
  • week6时期：争取万行python，拿下一个模型
  • week7全周集中攻克课程报告
  • week8课程报告各类结尾
  • week9-10分配时间进行托福集中攻克
  • week11-week12 托福封闭式学习
